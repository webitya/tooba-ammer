Type & Colour System

The single source of truth for every page — home, About, and the three case studies. One display serif set in italic, one body sans, and a six-value warm palette with a single oxblood accent. The discipline is the premium look; don't add more.

Every display style is Newsreader, weight 400, always italic, with optical sizing set so larger sizes tighten correctly. Sizes are written desktop-clamp → mobile. Negative tracking on headings; generous line-height on body.

Rules

  • One display serif (Newsreader), one body sans (Geist). Never add a third family, and never change them per page.
  • Newsreader is always set in italic for display and inline emphasis — that slant is the signature. Geist carries everything functional and stays upright.
  • The oxblood accent is for links, section numbers, eyebrow markers and hovers only. Never set body text in it — it fails contrast on paper and cheapens the page.
  • On dark (--ink) bands, switch text to the on-ink set and the accent to the warmer #C77449 for legibility.
  • Labels and eyebrows are uppercase Geist 500 with .18–.2em tracking. That wide tracking is the editorial tell — keep it consistent.
  • Headings get tight negative tracking; body stays normal with generous 1.65–1.7 line-height.
  • Captions are italic Newsreader; body copy is Geist. Don't swap them — the contrast is deliberate.
  • One Display XL per page. Step down the scale in order; don't jump from Display to Subsection.
  • Stat numerals stay on a single line — let them shrink before they wrap.
